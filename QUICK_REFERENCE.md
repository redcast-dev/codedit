## 🎯 How to Use AI Assistant

### **Step 1: Ask AI**
Type your request in the AI chat:
```
"Create a recipe generator app"
"Fix this bug"
"Refactor this code"
```

### **Step 2: Review Code**
AI responds with code in a highlighted block with:
- Blue border around code
- Blue banner at top: "AI generated code ready"

### **Step 3: Accept or Reject**
**Option A: Accept**
- Click green "Accept & Apply" button
- Code is applied to editor
- Saved to history for undo

**Option B: Reject**
- Click red "Reject" button
- Code is dismissed
- No changes made

### **Step 4: Revert if Needed**
- Click "Revert" button (↻) in AI panel header
- Undoes last accepted change
- Can undo multiple times

## 🔍 Using Search

1. Click Search icon (🔍) in activity bar
2. Type search term
3. Click "Search" button
4. Click any result to open file

## 🤖 Using AI Agents

1. Click AI Agents icon (🤖) in activity bar
2. Select agent:

   - Refactor Agent
   - Bug Fixer
3. AI chat opens with agent context
